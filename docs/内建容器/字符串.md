# 字符串
go语言中没有单个字节的char类型，在其他语言如Java，C++都是有这种类型的，这种类型在处理中文时会非常麻烦，go语言使用utf-编码，提供了4字节的rune类型作为字符串类型，可以避免复杂的中文编码问题。

## 字符串的一些常用操作
* 使用range进行字符串的遍历
* utf8.RuneCountInString获得rune字符数量
* len获取字符串长度-字节长度
* 使用[]byte获得字节
* Fields, Split, Join
* Contains, Index
* ToLower, ToUpper
* Trim, TrimRight, TrimLeft
